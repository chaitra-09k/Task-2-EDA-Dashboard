#  Task 2: Exploratory Data Analysis & Dashboard

##  Objective
Analyze the cleaned dataset to extract insights and present them using visualizations and a dashboard.

##  Steps Performed
- Performed Exploratory Data Analysis (EDA) using Python
- Created visualizations (bar, pie, line, scatter)
- Designed SQL queries to answer business questions
- Built a dashboard mockup using PowerPoint

##  Key Insights
- Sales are concentrated in a few major cities
- Customer distribution is dominated by one gender group
- Sales trends vary over time
- Higher sales do not always result in higher profit

##  Tools Used
- Python (Pandas, Matplotlib)
- SQL
- PowerPoint

##  Files Included
- EDA_code.py
- dashboard.pptx
- sql_queries.txt
- Charts (images)

##  Outcome
Developed a structured understanding of data analysis workflow from raw data to insights and visualization.
